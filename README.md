# GrupoVMPS
Repositório do nosso grupo para o projeto especificado na matéria de PS

---

### Montar pacote final para export (vem com dependências):

Windows: execute build-windows.bat

Linux: execute build-linux.sh

Executável e libraries estarão em ./target/z80emu/bin/

### Executar preview (não inclui dependencias)

Execute mvn javafx:run
